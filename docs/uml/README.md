# UML Diagrams - Sprint Documentation

This folder contains UML diagrams for User Stories 3 & 4.

1. **class-diagram.puml** - System class structure
2. **sequence-admin-application.puml** - Admin application submission flow
3. **sequence-admin-review.puml** - Admin review process
4. **sequence-vendor-registration.puml** - Vendor registration flow
5. **sequence-vendor-approval.puml** - Vendor approval process
6. **usecase-diagram.puml** - System use cases
7. **state-diagram.puml** - Application lifecycle states
8. **activity-diagram.puml** - Vendor registration activity flow
9. **component-diagram.puml** - System component architecture
10. **deployment-diagram.puml** - Deployment architecture

## How to View:

### Online:
1. Go to http://www.plantuml.com/plantuml/uml/
2. Copy and paste diagram code
3. View rendered diagram

### VS Code:
1. Install "PlantUML" extension
2. Open .puml file
3. Press Alt+D to preview

### Generate PNG/SVG:
```bash
# Install PlantUML locally
brew install plantuml  # Mac
# or
sudo apt-get install plantuml  # Linux

# Generate PNG
plantuml class-diagram.puml

# Generate SVG
plantuml -tsvg class-diagram.puml
```
